# monster-arena
Advanced Game Design Game

Controls

Keyboard:
WASD = Move
Move mouse = Move Camera
Hold Shift = Run
Space = Dodge
Left Click = attack
E = Use a health potion


Controller:
Left Joystick = Move
Right Joystick = Move Camera
R1 = Run
South Button = Dodge
Left Button = attack
North Button = Use a health potion

Cheats:
Press G to see damage numbers
Press R to restart the scene
